EcoChic

EcoChic is a React Native mobile app that empowers users to make sustainable fashion choices by scanning clothing tags and visualizing their environmental impact. Built with Expo Snack and integrated with the ASOS API, it calculates water usage, CO₂ emissions, and suggests greener material alternatives. I independently developed the UI with React hooks, implemented API integration, and created a custom sustainability scoring system with fallback mock data for offline use.
